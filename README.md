# How to achive Yolo Achivement

Here are the steps:

1. Create a new PUBLIC repository with README.md - https://github.com/new
2. Edit README.md and make a small change
3. Save README.md and create a new branch
4. Click Create pull request
5. **Important**: Instead of waiting for a code review, click Merge pull request directly without any review approvals
6. The YOLO achievement is unlocked when you merge a pull request without any code review

**important**: This will take some times so wait for it. try again.

## 🎉 Congratulations
